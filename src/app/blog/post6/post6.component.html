Agile is a set of principles and practices for software development that promotes adaptive planning, evolutionary development and delivery, a time-boxed iterative approach, and continuous improvement. Agile methods encourage rapid and flexible responses to change over following a plan.

DevOps is a methodology that improves communication between developers and IT operations staff. By working together, these teams can develop technology more efficiently: through continuous integration and delivery (CI/CD). CI refers to committing code changes into a source repository several times daily, while CD refers to deploying those changes automatically onto servers or virtual machines (VMs). Businesses can use CI/CD to update their software faster than ever to respond quickly to market conditions.
